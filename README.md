Enns, MS, PhD, Hawre J. Jalal, MD, PhD, M.G. Myriam Hunink, MD, PhD and Petros Pechlivanoglou, PhD.

- Appendix B: extended microsimulation model - Sick-Sicker example

- Appendix C: Markov cohort model - Sick-Sicker example
